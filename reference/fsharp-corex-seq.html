Iterates over elements of the input sequence and groups adjacent elements. A new group is started when the specified predicate holds about the element of the sequence (and at the beginning of the iteration).

For example: Seq.groupWhen isOdd [3;3;2;4;1;2] = seq [[3]; [3; 2; 4]; [1; 2]]
